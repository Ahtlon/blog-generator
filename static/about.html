About this Blog

This blog is my attempt to publish weekly articles on different topics ranging from technologies I'm currently working with such as Go, JavaScript or Docker to other more or less interesting things from the life of a Software Developer.

I hope you enjoy it! :)

About me

My name is Mario Zupan, I'm a Software Developer from Graz, Austria.

After finishing my MSc in Software Engineering & Economics at TU Graz I worked as a Full-Stack Web Developer for five years doing mostly Java and JavaScript with a focus on Clean Code and Software Craftsmanship.

In September 2016, I decided to quit my job and work as a freelancing consultant and trainer as well as to dedicate about half of my time to working in Open Source.

In the past, I organized the JavaScript User Group in Graz and I frequently visit and speak at other local meetups as well.

Currently I enjoy writing Go code and having the freedom to choose what to work on.

I'm always looking for interesting projects using Go or JavaScript - if you're looking for an experienced and motivated developer, feel free to contact me! :)
